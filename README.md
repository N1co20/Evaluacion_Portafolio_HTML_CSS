# Portafolio Web - Nicolas Ayala

## Descripción
Este proyecto consiste en un portafolio web desarrollado con HTML5 y CSS3.
En él se presenta información personal, habilidades, proyectos realizados y un formulario de contacto.

El objetivo es aplicar conocimientos de desarrollo web, utilizando etiquetas semánticas, estilos con CSS y organización del contenido mediante Flexbox y Grid.

-------------

## Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox
- CSS Grid

------------

## Estructura del sitio
El portafolio cuenta con las siguientes secciones:
- Inicio
- Sobre mí
- Habilidades
- Proyectos
- Contacto
